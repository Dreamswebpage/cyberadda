# Cyber Security Demo Website

A simulated cyber security dashboard that demonstrates attack detection with visual and audio alerts.

## Features
- Advanced cyber theme with animations
- Red light blinking during attacks
- Siren sound alert
- 60-minute countdown timer
- Network visualization
- Security logs
- Interactive controls

## Deployment on Netlify

1. **Create a Netlify account** (if you don't have one)
2. **Create a new site from Git**
3. **Connect your GitHub repository**
4. **Deploy!**

Or drag and drop the folder to Netlify Drop.
